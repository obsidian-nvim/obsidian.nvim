## Community Plugins

Close to the idea of community plugins for the app. They work in obsidian vaults, but they don't rely on vaults and work for normal markdown workflow.

- **WIP** [calendar.nvim](https://github.com/obsidian-nvim/calendar.nvim) for opening dailies, inserting dates, changing dates.
- [bulltes.nvim](https://github.com/kaymmm/bullets.nvim) not maintained, will fork it, inserts list markers and renumbers
- [kanban.nvim](https://github.com/arakkkkk/kanban.nvim/issues/36), _not yet vault aware_
- [quickadd.nvim](https://github.com/efirlus/quickadd.nvim) _not yet vault aware_
- [excalidraw.nvim](https://github.com/marcocofano/excalidraw.nvim) _not yet vault aware_

## General Markdown Utilities

- reanderers, previewers
- table-mode
- [otter.nvim](https://github.com/jmbuhr/otter.nvim) get LSP functionality in code blocks
- [markdowny.nvim](https://github.com/antonk52/markdowny.nvim) keymaps for adding links, bold, italic and inlines
- [hamidi-dev/org-list.nvim](https://github.com/hamidi-dev/org-list.nvim) orgmode-inspired, cycle through list states, make child parent checkboxes share states.
- [Thiago4532/mdmath.nvim](https://github.com/Thiago4532/mdmath.nvim) inline render markdown math
